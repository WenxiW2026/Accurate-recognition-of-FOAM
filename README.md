# Accurate-recognition-of-FOAM
Code of paper: Accurate recognition of fractional orbital angular momentum of light in complex scenarios using physics-informed deep learning

# Requirements:
Anaconda 3; Python 3.8.13; Pytorch 1.7.0; MATLAB

# How to use:
Step 1: Conduct experiments using patterns (pattern.mat) to record a series of intensities and save as a .mat file.

Step 2: Run './physics-informed deep learning.ipynb' to retrieve FOAM patterns in a dynamic, complex, disordered environment.

Step 3: Run './fractional_OAM_recognition.m' to recognize FOAM modes.
